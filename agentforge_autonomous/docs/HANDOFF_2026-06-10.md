## 6. Architecture Constraints

These apply to every file, agent, and output in this project. No exceptions.

| Constraint | Rule |
|---|---|
| Em-dashes | Forbidden. Use commas, colons, semicolons, or periods. |
| Modules | ES Modules only. No `require()`, no `module.exports`. |
| Package manager | npm only. Never yarn. |
| Deployment | Netlify only. Never Vercel. |
| LLM order | Ollama -> Groq -> Cerebras -> Together -> Mistral -> Claude -> OpenAI |
| Event model | Event-driven only. No polling loops inside agents. |
| Config | All values from `apps/job-agent/config/settings.json`. No hardcoded values. |
| Logging | ERROR/WARN/INFO/DEBUG structured logging only. No `console.log`. |
| Secrets | Never in git. `.env` files only. Never expose in documentation. |
| Audit rule | MISSING_EVIDENCE_REPORT before EXECUTIVE_SUMMARY. Always. |
